# 🔬 SpiceLab / 暧昧实验室

> 探索暧昧，实验爱情

私密情侣互动游戏平台 - 飞行棋、大转盘、真心话大冒险。

## ✨ 特性

- 🎲 **多种游戏模式** - 飞行棋、大转盘、骰子冒险
- 🎯 **双模式切换** - 基础版（随机真心话/大冒险）+ 高级版（自定义内容）
- 🔐 **隐私优先** - 支持自部署，数据在自己手里
- 🎨 **可定制** - 管理员自定义奖励/惩罚内容

## 📦 项目结构

```
SpiceLab/
├── packages/
│   ├── core/          # 游戏核心引擎
│   ├── ui/            # UI 组件库
│   └── templates/     # 题库模板格式
├── apps/
│   └── web/           # Web 应用
└── docs/              # 文档
```

## 🚀 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务
pnpm dev
```
